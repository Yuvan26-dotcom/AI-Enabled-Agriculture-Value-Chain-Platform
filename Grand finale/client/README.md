# Biometrix Frontend Client

This folder contains the frontend application for the Biometrix project.

## Tech Stack
- **Framework:** React (Vite)
- **Styling:** Tailwind CSS
- **Language:** JavaScript/JSX

## Prerequisites
- Node.js (v16 or higher)
- npm (Node Package Manager)

## Installation

1. Navigate to this folder in your terminal:
   ```bash
   cd client
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

## Running the Application

To start the development server:

```bash
npm run dev
```

The application will typically run on `http://localhost:5173` or `http://localhost:3000` (check the terminal output).

## Build for Production

To create a production build:

```bash
npm run build
```

The output will be in the `dist/` folder.
