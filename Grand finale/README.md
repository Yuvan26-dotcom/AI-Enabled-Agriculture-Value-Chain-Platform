# Unified Digital Agriculture Platform for Oilseed Supply Chain

This project integrates real-time IoT data, AI-driven decision support, blockchain traceability, and national Agri-Stack integration to optimize the oilseed supply chain.

## Project Structure

- **client/**: Frontend application (React/Vite)
- **server/**: Backend services (Node.js)
- **ai_service/**: AI Engine (Python)
